# AtCoder for Dart

- AtCoderのdartのバージョン: `3.0.5`

## プロジェクト作成

以下のコマンドで生成した後、不要なディレクトリやファイルは削除する。

```bash
dart create -t console [project]
```

削除するファイル・ディレクトリ

- 📁lib
    - 📄[project].dart
- 📄README.md
- 📄CHANGELOG.md
- 📄.gitignore

## テスト

```bash
dart run
```

## コンパイル(AtCoder)

```bash
dart compile exe bin/[project].dart
```
